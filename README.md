📊 Bank Loan Data Analysis
🔍 Comprehensive Analysis of Lending Patterns, Risk & Portfolio Performance
📌 Project Overview

This project focuses on analyzing a real-world bank loan dataset to uncover key insights into lending behavior, credit risk, and portfolio performance. The objective is to support data-driven decision-making in the banking sector by identifying patterns in borrower profiles, loan performance, and profitability.

🎯 Objectives
Analyze loan portfolio distribution and lending trends
Perform borrower profiling based on income, employment, and demographics
Evaluate loan performance (Fully Paid, Charged-Off, Current)
Identify risk indicators using DTI, interest rate, and loan grade
Assess the profitability and financial health of the portfolio
Explore loan purpose and geographic trends

🗂️ Dataset Summary
Total Records: 38,576 loan applications
Total Features: 24 columns

Key Variables:
Borrower Info: Income, Employment Length, Home Ownership
Loan Details: Loan Amount, Interest Rate, Term, Grade
Credit Metrics: DTI, Total Accounts
Performance: Loan Status, Total Payment

⚙️ Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (MySQL) for data querying
Excel for initial exploration
Power BI for interactive dashboard visualization

🧹 Data Cleaning & Feature Engineering
Handled missing values and validated dataset integrity
Converted date columns into proper datetime format
Standardized column names (snake_case)
Created new features:
Loan Category (Good vs Bad)
Income Segmentation (Low, Medium, High)
Time-based features (Month, Year, Weekday)

📊 Key Insights
💰 Portfolio Performance
Total Funded Amount: $435.76M
Total Received Amount: $473.07M
Net Profit: $37.31M → Indicates strong portfolio performance

📈 Loan Trends
Applications increased significantly throughout the year, with peak demand in Q4
Interest rates remained stable (~12%), ensuring a consistent pricing strategy

⚠️ Risk Analysis
13.8% loans are charged-off, contributing significantly to losses
DTI shows limited predictive power, indicating the need for advanced risk models

👤 Customer Insights
The majority of borrowers belong to the middle-income segment
High-income borrowers show better repayment behavior

🏦 Business Insight
The lending model is primarily volume-driven, where higher loan disbursement leads to increased revenue

📉 Dashboard
An interactive Power BI dashboard was developed to visualize:

Loan performance by status
Monthly trends and KPIs
Loan purpose analysis
Borrower segmentation

💡 Business Recommendations
Improve credit risk assessment models beyond DTI
Focus on high-performing loan categories (Debt Consolidation, Credit Card)
Reduce bad loans through stricter approval policies
Leverage seasonal demand trends (Q4 growth)
Enhance collection strategies to maximize profitability

🚀 Conclusion--
The analysis highlights that while the portfolio is profitable and growing, a significant portion of profit is impacted by bad loans. Implementing data-driven risk strategies and optimized lending policies can further enhance portfolio performance and sustainability.
